### Code Style
- **Python**: Follow PEP 8, use type hints
- **JavaScript/React**: Use ESLint configuration, functional components with hooks
- **API Endpoints**: Use kebab-case (e.g., `/start-conversation`)
- **Database Fields**: Use snake_case
